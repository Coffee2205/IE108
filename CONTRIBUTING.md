# Contributing

Thank you for contributing! Quick workflow:

1. Fork or clone the repo.
2. Create a feature branch: `git checkout -b feature/short-description`.
3. Install deps: `npm install`.
4. Run dev server: `npm run dev`.
5. Make commits (small, focused), push your branch, and open a PR against `main`.

SSH tips:
- Generate an SSH key: `ssh-keygen -t ed25519 -C "your_email@example.com"`
- Add the public key to GitHub (Settings → SSH and GPG keys).

If you need to match Node version, use `nvm use` (project `.nvmrc` contains recommended version).
